Different Types of Selectors

Simple Selectors

  • tag/element
    • uses the tag name to identify the element
    • This will affect all elements with that tag name
  • Class
    • uses the class name to style a group of elements
    • denoted by .className
  • ID
    • uses a unique id
    • denoted by #id
  • Pseudo-class
    • specific keyword at the end of the selector used to style elements when it is at a certain state
    • :active, :hover, etc...
  • Pseudo-element
    • used to style specific parts of an element
    • insert content before or after, style first letter, first line, etc.
    • ::before, ::after, ::firstline, ::firstletter

Combinator Selectors

  • Child Descendant
  • Sibling
  • Attribute Selectors
  • Multiple Selectors

Wild card selector

  • Just the *
  • Styles everything, all elements with the given rules
